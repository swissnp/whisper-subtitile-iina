Transcription Backend:

Choose whether to run transcriptions locally via whisper.cpp or use OpenAI’s cloud models (requires API key).

FFmpeg CLI Path:

Provide the absolute path to the ffmpeg binary. You may run brew install ffmpeg to install it.
If you are using Homebrew, the path is usually /opt/homebrew/bin/ffmpeg for Apple Sillion Macs and /usr/local/bin/ffmpeg for Intel Macs.

Whisper.cpp CLI Path:

Provide the absolute path to the whisper-cli binary. You may run brew install whisper-cpp to install it.
If you are using Homebrew, the path is usually /opt/homebrew/bin/whisper-cli for Apple Sillion Macs and /usr/local/bin/whisper-cli for Intel Macs.

Whisper.cpp Server Path:

Provide the absolute path to the whisper-server binary (built from whisper.cpp).

Whisper Server Connection:

Host/port where the plugin will connect to the server that is spawned automatically for each transcription.

Extra Whisper Server Options:

These options will be passed to whisper-server directly.

Subtitle Archive Directory:

Subtitles will be copied here after each run so they can be reused later. Supports @data, @tmp, etc.

OpenAI API Key:

Required when using the OpenAI backend. The key is stored locally inside IINA’s preferences.

OpenAI Model & Options:

Streaming with timestamps requires gpt-4o-transcribe-diarize and chunking_strategy=auto.

Audio uploads must be under 25 MB; the plugin will re-encode to a low bitrate MP3 automatically when necessary.

Only selected models will be shown in the subtitle choosing OSD.
